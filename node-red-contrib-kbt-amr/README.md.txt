# node-red-contrib-kbt-amr

Custom Node-RED nodes for KBT AMR based on SEER Robotics TCP API.

## Nodes

### AMR Control
- Movement
- Cancel Move
- Pause Move
- Resume Move
- Reset Alarm

### AMR Status
- Robot Speed
- Current POI
- Battery
- Alarm

### AMR Sound
- Sound Status

## Installation

npm install node-red-contrib-kbt-amr

Restart Node-RED
